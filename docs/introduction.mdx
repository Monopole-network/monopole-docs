[Monopole](https://mymonopole.com) is the first Startup studio solution matching Communities and ESG Projects for Impact Leveraging Blockchain technology with full transparency and decentralization

* **Monopole Protocol**: Persistent, non-upgradable [smart contracts](./smart-contracts/overview) that together create automated fundraising for environmental, social and economic commitment projects.
* **Monopole Interface**: A web interface that allows for easy interaction with [The Protocol](./protocol/overview).
* **Monopole API**: An [API](./api/overview) system linking updatable project metadata.
