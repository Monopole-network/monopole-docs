[Monopole](https://monopole.network) is the first global solution matching investors and projects for Impact Leveraging Blockchain technology with full transparency and decentralization

* **Monopole Protocol**: Persistent, non-upgradable [smart contracts](./smart-contracts/overview) that together create automated fundraising for impact projects.
* **Monopole Interface**: A web interface that allows for easy interaction with [The Protocol](./protocol/overview).
* **Monopole API**: An [API](./api/overview) system linking updatable project metadata.
