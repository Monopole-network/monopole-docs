Contains a subset of the full IMonopoleAllowlist interface that is used in Monopole

## Functions

### isAllowed

```solidity
function isAllowed(
    address user
) external view returns (bool);
```

Returns if a user is allowed

#### Parameters:

| Name      | Type    | Description                                                                    |
| :-------- | :------ | :----------------------------------------------------------------------------- |
| `user` | address | address of user |

#### Return Values:

| Type    | Description                          |
| :------ | :----------------------------------- |
| bool | user status |
